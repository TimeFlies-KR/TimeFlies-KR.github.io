index.html 편집기

방문자용 index.html에는 관리 버튼을 넣지 않고, 이 편집기에서 index.html을 불러와 글 목록만 수정합니다.

1. index.html 불러오기

Chrome 또는 Edge에서는 블로그 폴더를 선택하면 index.html을 읽고 저장까지 할 수 있습니다.
파일 선택 방식은 저장소에 직접 덮어쓰지는 못하고, 수정된 index.html을 다운로드합니다.

2. 글 추가

글 HTML 파일을 선택하면 파일명 기준으로 경로가 자동 입력됩니다. 예: Blog/과제.html

3. 저장

폴더에서 열었다면 “index.html에 저장”으로 바로 덮어쓸 수 있습니다. GitHub Pages에 반영하려면 이후 GitHub에 commit/push 해야 합니다.
