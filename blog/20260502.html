C# - 조건문

2026년 5월 2일

조건문의 종류는 if , else if , switch 등이 있다

if

if 는 조건이 참이라면 실행하고
if(조건)
{
실행문
}
이렇게 작성한다

else if

else if는 if랑 같이 사용하는데
if에서 조건이 거짓일때 새로운 조건을 붙여 검사 할때 사용한다
if(조건)
{
실행문
}
else if(조건)
{
실행문
}
이렇게 작성한다

switch

switch는 조건에 대한 값이 정해져 있는 경우에 많이 사용한다
switch(조건)
{
case 값 : 실행문
}

과제

과제 내용은
나이가 20세 이상이거나 쿠폰이 있으면 입장이 가능한 조건문을 완성하는것이다.
