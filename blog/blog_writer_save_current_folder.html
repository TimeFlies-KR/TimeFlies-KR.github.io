글 작성

현재 구조가 folder/과제.html, folder/css/base.css라면 현재 폴더 글 위치용을 선택하세요.
이미지
대표 이미지는 생성 HTML에 Base64로 포함됩니다. GitHub에서 이미지 파일로 따로 관리하려면 저장 후 직접 src를 바꿔도 됩니다.
본문에서 원하는 위치를 클릭한 뒤, 이미지의 “커서 위치에 삽입”을 누르면 그 자리에 들어갑니다.
이미지 문법 예시:
::image[사진 설명](image-abc){size=medium align=center}
파일이 생성되었습니다.
