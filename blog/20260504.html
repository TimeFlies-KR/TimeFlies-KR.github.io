C# - 변수와 데이터 타입

2026년 5월 4일

해당 과제에서 배운건 변수와 데이터 타입이지만
상수에 관해서도 써볼까 한다.

기본적으로 데이터를 담는 변수는 "데이터 타입 변수명 = 값;" 으로 정해진다
이러한 변수들은 "변수명 = 값;"으로 변경이 가능한데
"const 데이터 타입 변수명 = 값"
이런식으로 앞에 const를 붙혀 상수값으로 사용이 가능하다.

게임에선 "닉네임"같은것들은 변수명으로 사용하지만
"NPC이름"등은 고정으로 사용하기 때문에 상수값으로 사용하는게 좋다 .
왜냐하면 변수로 정해둘경우엔 변하면 안될 값마저 변할수 있기 때문이다.
ex) 다른개발자와의 소통미스 , 변수이름 실수

데이터 타입

기본적으로

숫자

int , long, float, double

문자

char

문자열

string

논리

bool 등이 주로 사용된다
