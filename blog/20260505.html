C# – 연산자

2026년 5월 5일

연산자는 산술 연산자, 비교 연산자, 논리 연산자 3종류가 있다.

산술연산자

산술 연산자는 숫자의 연산을 담당한다
ex) 덧셈, 뺄셈

연산자 종류는

  1. + (더하기)
  2. - (빼기)
  3. / (나누기)
  4. * (곱하기)
  5. % (나머지)

    6. 가 있다

다른건 다 쉬운데 5번 나머지는 감이 안온다면 나누었을때 남은 나머지 값을 반환한다
ex) 7 % 3 = 1 , 6 % 3 = 0

비교연산자

비교 연산자는 값을 비교하여 참인지 거짓인지 판단한다
연산자 종류는

  1. == (같다)
  2. != (같지 않다)
  3. > (크다)
  4. < (작다)
  5. >= (크거나 같다)
  6. <= (작거나 같다)

    7. 가 있다

논리 연산자

논리 연산자는 조건을 연결해주는 역할을 하는데 연산자 종류를 보면 감이 온다

  1. && (그리고)
  2. || (또는)
  3. ! (아니다)

    4. 가 있다

과제

과제 내용은
아래 조건에 대해 true 값이 나오는 연산식을 완성하세요.

어떤 사람의 나이(age)가 65세 이상이거나, 또는 VIP회원(isVip)이면 우대 대상입니다.

①, ②에 맞는 연산자를 작성하세요.

age ( ① ) 65 ( ② ) isVip == true
이다

정답은 >= , || 이다
