C# - 반복문

2026년 5월 3일

반복문의 종류는 for, while, foreach 등이 있다

for

for은 "특정 횟수만큼 반복해줘" 인데
기본 형태는

for(int i =0; i <= 5; i++)
{
실행문
}
이렇게 작성한다

while

while은 "조건이 참일때 계속 반복해줘" 이고
기본 형태는
while(조건)
{
실행문
}
이렇게 작성한다

조건을 거짓으로 만드는 코드를 중간에 넣지 않거나
실수로 거짓으로 만들지 못했을 경우 프로그램이 멈춘다.
이 실수가 유니티 프로그램을 가장 많이 멈추게 했던거 같다.

foreach

foreach는 배열이나 리스트에서 값을 꺼내서 반복한다
기본형태는
string[] color = {"Red","Blue","Green"}
foreach(string 이름 in 배열 이름)

{
Console.WriteLine(이름);
}
이렇게 작성한다
